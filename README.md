# 碧潭能源管理系統 PRO

水電瓦斯抄錶、用量趨勢分析與異常警示的單頁式管理工具。純前端 HTML/CSS/JS 打造，無需伺服器、無需安裝，資料全部儲存在使用者裝置的瀏覽器（`localStorage`）中，可作為 PWA 加到手機主畫面離線使用。

## 功能

- **多站點抄錶管理**：新增／編輯／刪除站點，記錄群組、位置、錶號
- **水／電／瓦斯**三種類型，自動計算「本期 − 上期」用量，並即時驗證輸入是否合理（讀數倒退、用量暴增會跳出提醒）
- **月份切換**：依月份記錄抄錶讀數，自動帶入上月讀數作為本期上期
- **拍照存證**：可直接呼叫手機相機拍攝錶況照片
- **圖表分析**：近 6／12 期用量趨勢長條圖、水電瓦斯佔比圓餅圖、異常用量警示、用量排行榜，並可依群組篩選
- **報表匯出**：本期報表匯出 CSV（Excel 可直接開啟）、列印 / 匯出 PDF、圖表分析報告匯出
- **資料備份**：匯出／匯入完整 JSON 備份
- **深色模式**：適合夜間抄錶使用
- **響應式版面**：手機、平板、桌機皆自動適配

## 快速開始

### 本機開啟

直接用瀏覽器打開 `index.html` 即可使用，不需要安裝任何套件或伺服器。

### 部署到 GitHub Pages（建議）

1. 將本專案 push 到 GitHub repository
2. 進入 repository 的 **Settings → Pages**
3. Source 選擇 `main` 分支、`/ (root)` 目錄，儲存
4. 幾分鐘後即可透過 `https://<你的帳號>.github.io/<repo名稱>/` 存取

### 加到手機主畫面（PWA）

**iPhone (Safari)**
1. 用 Safari 開啟部署後的網址
2. 點選分享按鈕 → 「加入主畫面」
3. 之後從主畫面圖示開啟即為全螢幕獨立 App 體驗

**Android (Chrome)**
1. 用 Chrome 開啟部署後的網址
2. 選單 → 「安裝應用程式」或「加到主畫面」

## 資料儲存說明

所有資料（站點清單、每月抄錶讀數、照片、深色模式偏好）都儲存在瀏覽器的 `localStorage`，**僅存在於單一裝置、單一瀏覽器**，不會自動跨裝置同步。

- 換裝置或清除瀏覽器資料前，請先用「選單 → 匯出完整備份」下載 JSON 備份檔
- 需要換到新裝置時，用「選單 → 匯入備份還原」即可還原所有資料

## 技術說明

- 純原生 HTML / CSS / JavaScript，無任何外部框架或建置流程
- 圖表使用 Canvas 2D 手繪，不依賴外部圖表函式庫，可離線運作
- 無後端、無資料庫、無 API，所有運算與儲存都在瀏覽器端完成
